# Honeysmile's Hunter Companion

A single-page, offline-friendly companion for **Monster Hunter Wilds**: a
**Builds** planner (armor, decorations, charms, set/group skills, Transcend
slots, shareable build codes), plus monster weaknesses & hitzones, gathering,
trade, weather, and consumable crafting.

## Files

```
index.html        The app (open directly in any browser)
armor.js          Builds planner — sets, skills, slots, Transcend, build codes, presets
decorations.js    Decoration data + slotting
charms.js         Charm data
support.js        Shared helpers
image-slot.js     Image drop-slot component
images/           Icons, items, materials, monster art
```

## View it

- **Locally:** open `index.html` in any modern browser.
- **GitHub Pages:** push this folder, then *Settings → Pages → Branch: main / root*.
  Served at `https://<your-username>.github.io/<repo>/`.

## Features

- **Builds** — mix pieces across sets; auto-totals skills (current/max, over-cap
  warning), tracks Set Bonus + Group Skills, decoration slotting, charms,
  per-piece **Transcend** slot toggle, rarity-colored icons, search/filter,
  curated preset builds, and short shareable **build codes**
- **Monsters** — weaknesses, status susceptibility, per-part hitzones
- **Gathering / Trade / Weather** — region loot, trader exchanges, conditions
- **Consumable crafting** — checklist + auto shopping list (expands sub-recipes,
  labels each raw material with what it's *for*)
